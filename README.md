# Projet---Sujet-projet-power4
Ce projet consiste en la réalisation **d'un jeu de Puissance 4** multijoueur ***local*** avec une interface web, développé en Go, HTML et CSS.<p>
##  Objectifs:
- Implémenter un jeu de Puissance 4 respectant les règles classiques.<p>
- Développer une interface web intuitive avec un serveur HTTP en Go.<p>
- Permettre à deux joueurs de s'affronter sur la même machine.<p>
## Fonctionnalités:
- **Page d'accueil** (`/templates/accueil`) : Présentation du projet et règles du jeu<p>
- **Initialisation** (`/templates/init`) : Saisie des pseudos et choix des couleurs<p>
- **Jeu** (`/templates/play`) : Interface de jeu avec grille interactive<p>
- **Fin de partie** (`/templates/end`) : Affichage du résultat avec option "Rejouer"<p>
- **Scoreboard** (`/templates/history`) : Historique des parties jouées<p>

## Installation et exécution:
- **Backend** : Go (serveur HTTP, templates HTML)<p>
- **Frontend** : HTML, CSS<p>
